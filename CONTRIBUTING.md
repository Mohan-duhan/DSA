## How to Contribute

### 1. Fork the Repository
Click the "Fork" button at the top right of this page.

### 2. Clone Your Fork
```bash
git clone https://github.com/YOUR_USERNAME/DSA_LC.git
cd DSA_LC
```

### 3. Create a Branch
```bash
git checkout -b feature/your-solution
```

### 4. Add Your Solution

- Create a folder for the problem (e.g., `two-sum/`)
- Add your solution file with proper naming
- Include comments explaining your approach
- Add time and space complexity analysis

### 5. Commit Your Changes
```bash
git add .
git commit -m "Add: [Problem Name] solution in [Language]"
```

### 6. Push to Your Fork
```bash
git push origin feature/your-solution
```

### 7. Create a Pull Request
Go to the original repository and click "New Pull Request"
